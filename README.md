## Setup

> - You might need `GNU GCC`, `GNU GDB`, and `GNU Make`.
> - All the programs here runs and tested in `Debian Linux`.

```bash
# Generate an executable file.
make build

# Run the program inside GDB.
make debug

# Run the program normally.
make run
```
